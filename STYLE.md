# HTML Style Guide — v2

Reference for the lesson template. Every lesson generated by `/teachme` must follow this spec.

---

## Fonts

Load via a single Google Fonts `<link>` in `<head>`:

```
https://fonts.googleapis.com/css2?family=Merriweather:ital,wght@0,400;0,700;1,400
  &family=Playfair+Display:ital,wght@0,600;0,700;1,600
  &family=Inter:wght@400;500;600
  &family=JetBrains+Mono:ital,wght@0,400;0,500;1,400
  &display=swap
```

| Role | Font | Size | Line-height | Notes |
|---|---|---|---|---|
| Body copy | Merriweather | 17.5px | 1.85 | Large x-height, low eye strain |
| h1 (lesson title) | Playfair Display | 2.2rem | 1.2 | Bold 700, letter-spacing -0.01em |
| h2 (section heading) | Playfair Display | 1.2rem | — | Semibold 600, letter-spacing -0.01em |
| Labels / captions / UI | Inter | 10–14px | 1.5 | Use for all small text and table headers |
| Code / monospace | JetBrains Mono | 0.81rem | 1.7 | Use in all code contexts |

## Colour Tokens

Defined as CSS custom properties on `:root`. Use tokens — never raw hex — in all component styles.

```css
:root {
  /* Page base */
  --page-bg:      #faf8f4;   /* warm cream */
  --ink:          #1c1917;   /* near-black body text */
  --ink-2:        #44403c;   /* paragraph text */
  --ink-muted:    #78716c;   /* subtitles, captions */
  --warm-rule:    #e7dece;   /* dividers, borders */
  --warm-accent:  #c4a882;   /* tan — labels, eyebrows, footer */
  --warm-dark:    #2c1f14;   /* espresso — summary bg */
```

## Page Structure

```html
<body>
  <div class="running-header">...</div>        <!-- chapter · topic / course name -->

  <div class="lesson-eyebrow">Lesson 01</div>
  <h1>Lesson Title</h1>
  <p class="lesson-subtitle">One-line italic subtitle.</p>

  <h2>Section</h2>
  <p>Body copy...</p>

  <!-- content blocks (order flexible) -->

  <div class="tree-locator">...</div>          <!-- always before summary -->
  <div class="summary-block">...</div>         <!-- always last before footer -->

  <div class="page-footer">...</div>
</body>
```

Layout: single column, `max-width: 720px`, `margin: 0 auto`, `padding: 0 48px 96px`.

## Components

### Running Header
```html
<div class="running-header">
  <span class="rh-left">Chapter N · Section Name</span>
  <div class="rh-rule"></div>
  <span class="rh-right">Course Name</span>
</div>
```
Inter 10px, 600 weight, uppercase, `--warm-accent`. Separated by a `--warm-rule` flex line.

---

### Definition Callout
```html
<div class="definition">
  <div class="definition-label">Definition N.N — Term</div>
  <p>Body of definition.</p>
</div>
```
Warm cream bg, `--warm-rule` border, `--warm-accent` left rule (3px). Used inline within body text.

---

### Code Block · cool blue
```html
<div class="code-block">
  <div class="code-block-header">
    <span>language · description</span>
    <span>Listing N.N</span>
  </div>
  <pre>... syntax-highlighted code ...</pre>
</div>
```
Background `--code-bg`, border `--code-border`, header `--code-header`.

**Syntax highlight classes:**
| Class | Colour | Use |
|---|---|---|
| `.syn-kw` | `#9a3412` | keywords, variable names |
| `.syn-fn` | `#1e5fa8` | function names |
| `.syn-num` | `#7c3aed` | numbers |
| `.syn-str` | `#166534` | strings |
| `.syn-cm` | `#7a9ab8` italic | comments |

---

### Equation Block · sage green
```html
<div class="equation-block">
  <div class="equation-block-label">Equation N.N — Title</div>
  \[ LaTeX equation here \]
  <span class="equation-tag">(N.N)</span>
  <div class="equation-caption">Caption text.</div>
</div>
```
Background `--eq-bg`, border `--eq-border`. Requires MathJax 3 CDN.

---

### Figure Block · dusty rose
```html
<figure class="figure-block">
  <div class="figure-placeholder">
    <!-- or <img src="..."> -->
    <span>Description of figure</span>
  </div>
  <figcaption><strong>Figure N.N</strong> — Caption text.</figcaption>
</figure>
```
Block frame: `--fig-bg`, border `--fig-border`. Caption strip attached below image, no top border.

Content colours are **flexible** — diagrams, SVGs, and images use whatever colours suit the subject being depicted. The dusty rose frame is fixed; what goes inside it is not.

---

### Graph Block · amber
```html
<div class="graph-block">
  <div class="graph-header">
    <span>Figure N.N · Title</span>
    <span>chart type</span>
  </div>
  <div class="graph-canvas-wrap">
    <canvas id="chart" height="180"></canvas>
  </div>
  <div class="graph-caption"><strong>Figure N.N</strong> — Caption.</div>
</div>
```
Block frame: `--graph-bg`, border `--graph-border`. Requires Chart.js 4.4 CDN.
Chart.js tick font: `Inter`, colour `--graph-label`. Grid colour `#ede8ce`.

Dataset colours (bar fills, line strokes, area fills) are **flexible** — choose colours that match the subject matter of the chart. The amber frame is fixed; the data painted inside it is not.

---

### Quiz Block · soft lavender
```html
<div class="quiz-block" data-explanation="Explanation shown after any answer.">
  <div class="quiz-block-label">Quiz N.N — Topic</div>
  <p class="quiz-question">Question text?</p>
  <div class="quiz-options">
    <!-- All options must be equal in word count and character length where possible -->
    <button class="quiz-option" data-correct="false">Option A</button>
    <button class="quiz-option" data-correct="true">Option B</button>
    <button class="quiz-option" data-correct="false">Option C</button>
    <button class="quiz-option" data-correct="false">Option D</button>
  </div>
  <div class="quiz-feedback"></div>
</div>
```
Background `--quiz-bg`, border `--quiz-border`. Requires quiz JS (see External Dependencies).

After selection: correct option highlighted green, wrong selection highlighted red, all buttons disabled. The `data-explanation` on the container is appended to the feedback message.

---

### Practice Block · soft teal
```html
<div class="practice-block">
  <div class="practice-block-label">Practice N.N — Title</div>
  <ol class="practice-steps">
    <li>
      <label class="practice-step">
        <input type="checkbox">
        <span>Step description. <code>inline code</code> supported.</span>
      </label>
    </li>
  </ol>
</div>
```
Background `--prac-bg`, border `--prac-border`. Checked steps get struck-through text. Requires practice JS (see External Dependencies).

---

### Tree Locator · warm tan
```html
<div class="tree-locator">
  <div class="tree-locator-label">Where this lesson sits</div>
  <div class="tree-row">
    <span class="tree-key">Branches from</span>
    <span class="tree-val">The concept or trunk this lesson builds on.</span>
  </div>
  <div class="tree-row">
    <span class="tree-key">Unlocks next</span>
    <span class="tree-val">Concept A · Concept B · Concept C</span>
  </div>
</div>
```
Uses `--warm-rule` borders and `--warm-accent` label. Always placed between the last body section and the summary block.

---

### Table
```html
<table>
  <thead>
    <tr><th>Column A</th><th>Column B</th><th>Column C</th></tr>
  </thead>
  <tbody>
    <tr><td>Cell</td><td>Cell</td><td>Cell</td></tr>
  </tbody>
</table>
```
Full-width, `border-collapse: collapse`. Headers: Inter uppercase, `--ink-muted`. Rows: `--warm-rule` dividers, subtle hover.

---

### Summary Block · espresso inverted
```html
<div class="summary-block">
  <div class="summary-block-label">End of Lesson NN</div>
  <h3>Key Points to Remember</h3>
  <ul>
    <li>
      <span class="sum-n">01</span>
      <span>Point text. <code>inline code</code> supported.</span>
    </li>
  </ul>
</div>
```
Background `--sum-bg` (dark). Text `--sum-ink`. Numbers `--sum-muted`. Inline `<code>` uses translucent bg. Always the last content block before the footer.

## External Dependencies

Add to every lesson `<head>` in this order:

```html
<!-- Fonts (always) -->
<link href="https://fonts.googleapis.com/css2?family=Merriweather:ital,wght@0,400;0,700;1,400&family=Playfair+Display:ital,wght@0,600;0,700;1,600&family=Inter:wght@400;500;600&family=JetBrains+Mono:ital,wght@0,400;0,500;1,400&display=swap" rel="stylesheet">
<!-- Equations (only if lesson contains equations) -->
<script src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-svg.js"></script>
<!-- Charts (only if lesson contains graphs) -->
<script src="https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js"></script>
```

Only include MathJax and Chart.js in lessons that use them.

**Quiz JS** — include in `<script>` at end of `<body>` in any lesson with a quiz block:

```javascript
document.querySelectorAll('.quiz-block').forEach(function(quiz) {
  var explanation = quiz.dataset.explanation || '';
  quiz.querySelectorAll('.quiz-option').forEach(function(btn) {
    btn.addEventListener('click', function() {
      var correct = btn.dataset.correct === 'true';
      quiz.querySelectorAll('.quiz-option').forEach(function(b) {
        b.disabled = true;
        if (b.dataset.correct === 'true') b.classList.add('opt-correct');
        else if (b === btn) b.classList.add('opt-wrong');
      });
      var fb = quiz.querySelector('.quiz-feedback');
      fb.className = 'quiz-feedback ' + (correct ? 'feedback-correct' : 'feedback-wrong');
      fb.textContent = (correct ? '✓ Correct. ' : '✗ Not quite — the correct answer is highlighted. ') + explanation;
    });
  });
});
```

**Practice JS** — include in `<script>` at end of `<body>` in any lesson with a practice block:

```javascript
document.querySelectorAll('.practice-step').forEach(function(label) {
  label.querySelector('input[type="checkbox"]').addEventListener('change', function() {
    label.classList.toggle('done', this.checked);
  });
});
```

## Print

All lessons should print cleanly. Add this media query at the end of the `<style>` block:

```css
@media print {
  body { font-size: 15px; padding: 0 24px 48px; }
  .running-header { position: static; }
  .code-block, .equation-block, .figure-block, .graph-block,
  .quiz-block, .practice-block, .summary-block,
  .tree-locator { page-break-inside: avoid; }
  .page-footer { page-break-inside: avoid; }
  .quiz-option { opacity: 1 !important; }
}
```
